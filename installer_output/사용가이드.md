# 🎬 Whisper 배치 전사 도구 — 사용 가이드

## 🎵 지원하는 파일 형식

| 형식   | 설명        | 비고                         |
| ------ | ----------- | ---------------------------- |
| `.mkv` | 동영상 파일 | ffmpeg로 오디오 추출 후 전사 |
| `.wav` | 오디오 파일 | 직접 전사 (가장 빠름)        |

> 💡 **MP4, MP3, M4A 등 다른 형식**은 아직 지원하지 않습니다.
> 다른 형식의 파일은 먼저 WAV로 변환 후 사용해주세요.

---

## 📦 설치하기

1. **`WhisperBatchTool_Setup.exe`**를 더블클릭합니다
2. 화면 안내에 따라 **다음 → 다음 → 설치** 를 클릭합니다
3. 설치가 끝나면 자동으로 환경 구성이 진행됩니다 _(약 5~10분 소요)_

> ⚠️ **사전 준비물**
>
> - 인터넷 연결 (필수)
> - [Python](https://www.python.org/downloads/) 설치 _(설치 시 "Add Python to PATH" 반드시 체크!)_
> - [ffmpeg](https://ffmpeg.org/download.html) 설치 _(MKV 파일 처리 시 필요)_
>   - Windows 11: 터미널에서 `winget install ffmpeg`

---

## 🚀 사용하기

### 1단계: 프로그램 실행

바탕화면의 **"Whisper 전사 도구"** 아이콘을 더블클릭합니다.

### 2단계: 파일 형식 선택

```
🎯 작업 대상 선택:
  1: MKV 파일
  2: WAV 파일
```

### 3단계: 전사할 파일 선택

```
📂 파일 선택 방법:
  1: 📁 파일 탐색기에서 선택 (추천)  ← 가장 편리!
  2: 📋 폴더 경로 직접 입력
  3: 📂 현재 폴더에서 자동 검색
```

**1번을 선택하면** 파일 탐색기가 열리고, 원하는 파일을 골라서 선택할 수 있습니다.
여러 파일을 한번에 선택하려면 **Ctrl+클릭** 또는 **Shift+클릭**을 사용하세요.

### 4단계: 모델 선택

```
🤖 Whisper 모델 선택:
  1: tiny    — 매우 빠름, 품질 낮음
  2: base    — 빠름, 보통 품질
  3: small   — 보통, 높은 품질 ⭐ 추천
  4: medium  — 느림, 매우 높은 품질
  5: large   — 매우 느림, 최고 품질
```

> 💡 처음 사용할 때는 **2 (base)** 또는 **3 (small)** 을 추천합니다.

### 5단계: 결과 확인

전사가 완료되면 **원본 파일과 같은 폴더**에 `.txt` 파일이 생성됩니다.

예시: `회의녹음.mkv` → `회의녹음.txt`

## ❓ 자주 묻는 질문

**Q: 처리 시간이 얼마나 걸리나요?**

- 1시간 분량 기준: tiny 약 3분, base 약 5분, small 약 15분 (PC 성능에 따라 다름)

**Q: GPU가 있으면 더 빨라지나요?**

- 네! 설치 폴더의 `install.bat`을 실행하고 GPU 버전을 선택하면 5~10배 빨라집니다.

**Q: 영어 영상도 전사 가능한가요?**

- 현재는 한국어 전용입니다. 다른 언어가 필요하면 코드를 수정해야 합니다.

**Q: 설치 후 실행이 안 됩니다**

- Python이 설치되어 있는지 확인하세요
- 설치 폴더의 `install.bat`을 다시 실행해보세요
